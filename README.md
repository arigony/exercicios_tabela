# Química Geral - Lista 3 Interativa

Página HTML única para exercícios interativos sobre propriedades periódicas.

## Como publicar no GitHub Pages

1. Crie um repositório no GitHub.
2. Envie o arquivo `index.html` para a raiz do repositório.
3. Vá em **Settings > Pages**.
4. Em **Build and deployment**, selecione **Deploy from a branch**.
5. Escolha a branch `main` e a pasta `/root`.
6. Aguarde o link do GitHub Pages.

## Como usar no Moodle

- Envie o `index.html` como arquivo ou incorpore em uma página do Moodle.
- A página é responsiva e foi pensada para celular.
- O ranking é local: fica salvo apenas no navegador/dispositivo do aluno.

## Recursos

- 10 questões interativas.
- Feedback imediato.
- Explicação para acertos e erros.
- Pontuação automática.
- Ranking local.
- Botão de impressão/salvar em PDF.
- Sem bibliotecas externas.
